MediCare API

Опис проєкту



MediCare API - це високопродуктивний, відмовостійкий бекенд-сервіс, створений спеціально для сфери медичного обслуговування. Головна мета проєкту - забезпечити надійну й безпечну інтеграцію між медичними установами, управління електронними картками пацієнтів та автоматизацію клінічних процесів.



Інструкція зі встановлення

Для запуску проєкту на вашому локальному комп'ютері мають бути встановлені:

Erlang Elixir PostgreSQL

Способи використання проєкту

Для детального ознайомлення з усіма аспектами платформи ми пропонуємо три окремі види документації залежно від ваших цілей:

Технічні специфікації: Архітектурні рішення, схема БД у PostgreSQL, дерево супервізії процесів в Elixir.

Документація Production API: Довідник ендпоінтів, форматів JSON-запитів, відповідей та автентифікації для розробників інтеграцій.

Бізнес-процеси та специфікації: Опис логіки надання медичних послуг, взаємодії суб'єктів та бізнес-правил системи.
